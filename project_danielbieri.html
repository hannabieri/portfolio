Minimalistic web appearance for an architecture office

My role Webdesign • Front-End Development
Context Side project

THE STARTING POINT

Having an up-to-date web appearance became standard in our time, but especially for a self-employed architect, it's crucial to have a good portfolio website to showcase their work. In this project, it was important to find a way for a fresh, but genuine and not „mainstream“ way of building a new visual appearance for Daniel Bieri. Thus, a central challenge was to find a style which suits his architecture and therefore represents him in the best way possible. The client's main goal was to develop a site that would give the architecture a lot of space and thus create a focus on his works.

Design • WEB DESIGN

With its minimalistic and clean style, the design creates a lot of room for the content to take effect. Also, it fits well with the architectural style of Daniel Bieri. It’s a responsive web design, focusing on mobile-first. Therefore, the usability on phone screens provides a seamless experience to its users. It’s easy to flip from one project to another project and get a feeling for the style of architecture. But also in a web browser, the website appears well-balanced thanks to a responsive grid which has been used. Like this, the user can entirely focus on the architect and his works, without getting distracted by anything.

IMPLEMENTATION • FRONT-END DEV

After creating and iterating the design of the site, I also realized the project. The front-end implementation was mainly done with HTML5 and CSS3, occasional use of JavaScript, too. It’s a responsive page with three breakpoints and different designs adapted to the screen size of the according device.
